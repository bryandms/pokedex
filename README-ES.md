## Instalar

**Paso 1:** Clone o descargue el repositorio.

**Paso 2:** Cambie al directorio de su repositorio.

```bash
cd /ruta/de/su/repo
```

**Paso 3:** Instale las dependencias.

```bash
# NPM
npm i

# Yarn
yarn
```

**Paso 4:** Inicie el proyecto.

```bash
# NPM
npm run android
npm run ios

# Yarn
yarn android
yarn ios
```
